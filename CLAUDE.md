# 有明帰宅ナビ — Claude Code 引継ぎドキュメント

## プロジェクトの目的
都内の任意の場所と Brillia有明スカイタワー（江東区有明）の間の移動ルートを、
確定済みの効用関数で即時判定する個人用アプリ。主ユースケースは深夜の会食帰り（帰宅）。
v0.5でおでかけ方向（有明→都内）を追加（2026-07-14オーナー指示）。

## 効用関数（確定仕様・変更禁止、変更時はオーナー承認必須）
※効用関数自体は両方向共通に適用する（v0.5、2026-07-14オーナー承認）
1. ハード制約: タクシー支出 ≤ 上限額/回（超過選択肢は非推奨、表示はする）
   - 上限額はUIで500円刻み可変（¥500〜¥10,000、既定¥3,000、localStorage保存）。
     v0.4でオーナー指示により固定¥3,000から可変化（2026-07-13承認）
2. 制約内で door-to-door 時間最小（起点駅改札→自宅玄関）
3. 時間差 ±5分以内はタイブレークで安い方
- 深夜割増: 22:00–4:59 = 2割増（現行: 初乗¥500/1km、加算¥100/255m。2026年10月に改定予定あり→要追随）
- 夕ラッシュ(平日17–20時): 直行タクシー所要上限×1.5、料金+10%
- 終電後は鉄道系選択肢を無効化し、上限超過警告付きで最安タクシー案を提示

## アーキテクチャ（index.html内）
- `ZONES` … 手作業較正済みの高精度ゾーン25件（主要繁華街）。マッチング最優先。
- `METRO` … 18路線（メトロ9・JR3・都営4・ゆりかもめ・りんかい線）の駅順序＋ハブ定義。
  `genStationZone()` が任意の駅の選択肢を計算生成（名寄せ後250駅）。
  ループ路線対応（山手線・大江戸線環状部）、乗換ハブ対応（千代田/南北/副都心/総武/新宿線）。
  ゆりかもめ・りんかい線は `transit.direct:true` の直乗りモデル（v0.3）:
  有明駅（徒歩4分）/国際展示場駅（徒歩6分）に直乗り→下車徒歩。ハブ乗換なし。
- `HUBDEF` … ハブ駅マスタ: 豊洲/勝どき/門前仲町/清澄白河/築地/新橋/銀座/田町/潮見。
  設計原則:「ラストワンマイルが短いハブに乗換ゼロで着けるか」が判定の核。
- `headwayOf()` … 路線×時間帯の運行間隔モデル（静的）。待ち超過分を所要上限に加算。
- `buildOptions(zone, timeMin, weekend, dir)` / `recommend()` … 効用関数の実装本体。
  `dir:"home"|"out"` で帰宅/おでかけ両方向を生成（v0.5）。ハブ・transitは同一データを対称利用。
- バス統合（v0.5）: `BUSROUTE`（東京BRT幹線 ¥220 / 都05-2 ¥210 / 海01 ¥210 の乗車分・
  徒歩分・終発 wd/we・運行間隔）、`HUBBUS`（ハブ駅接続バス→「鉄道×バス乗継」タクシー0円案）、
  `CITY_ANCH`+`zone.city`（新橋/虎ノ門ヒルズBRT乗降点⇔目的地のタクシー接続。curatedは手動km、
  生成駅は路線トポロジから0.9km/駅で推定）。ゾーンの `bus:[{r,w}]` がバス直行案。
  自宅側停留所: BRT=有明テニスの森（徒歩7分・要較正）、都05-2=有明一丁目（徒歩4分）。
- AI地名解決 … api.anthropic.com 直叩き。**Claude.ai外では動作しない（既知）**。
  スタンドアロン環境では辞書231駅のみでフォールバック動作。

## 既知の限界（較正待ち）
- 生成駅の直行タクシー料金は ±15% 推定（dirCoef係数＋OVERRIDE_KMで較正中）
- ゆりかもめ・りんかい線（v0.3追加）: 運賃は fareRail+addFare 近似（実額と±¥80程度）、
  終電は路線一律（ゆりかもめ23:35/りんかい23:45）。台場エリアのタクシー距離は橋・運河迂回を
  OVERRIDE_KM で概算済みだが実測未較正
- 終電時刻は一律近似（23:35〜24:10）。実ダイヤ未反映
- 乗換所要は駅平均値（大江戸線の深さ等の個体差は未反映）
- 私鉄（東急・小田急・京王・西武・東武）未収録
- バス（v0.5追加）: 所要・終発は静的目安。確認済み実データ＝BRT有明テニスの森発新橋行き
  最終 平日22:37/土休22:16・新橋→有明約15分・都05-2東京駅発最終22:50（有明一丁目行き）。
  未確認＝新橋発BRT最終（22:50と仮置き）、都05-2有明発最終（22:00と仮置き）、海01終発
  （21:30-21:45と保守的仮置き）、虎ノ門ヒルズ直通B05の本数・終発、自宅→有明テニスの森
  停留所の徒歩分（7分と仮置き）。オーナー実測で較正する
- おでかけ方向の終電・終バスは有明側発の近似値。ハブ鉄道の最終は帰宅用の値を流用（近似）
- オーナーからの実測フィードバックで OVERRIDE_KM / lo,hi / headwayOf を随時較正する運用
